# GDA Finance Platform v8.2 🚀

GDA Finance, kurumsal finans yönetimi, faturalandırma ve personel takip süreçlerini modernize eden, **Vite** tabanlı yüksek performanslı bir web uygulamasıdır. Monolitik bir yapıdan profesyonel modüler mimariye dönüştürülmüştür.

## ✨ Özellikler

- **Gelişmiş Dashboard**: Gelir, gider ve kâr marjlarını anlık takip eden görsel grafikler (Chart.js).
- **Fatura Editörü**: 10 farklı profesyonel şablon, renk ve font özelleştirme, canlı önizleme.
- **Modüler Finans Modülleri**:
  - Gelir & Gider Takibi (Belge/OCR Altyapısı)
  - Müşteri Yönetimi (Kurumsal & Bireysel)
  - Personel & Bordro (Maaş hesaplama motoru)
  - Ürün & Hizmet Kataloğu
- **Profesyonel Altyapı**:
  - SPA (Single Page Application) Router
  - Merkezi State Yönetimi & Veri Kalıcılığı
  - Global Tema & Modern UI Tasarımı

## 🛠️ Kurulum

Projeyi yerelinizde çalıştırmak için:

```bash
# Bağımlılıkları kurun
npm install

# Geliştirme sunucusunu başlatın
npm run dev

# Production build oluşturun
npm run build
```

## 🌐 Canlı Yayın (Deployment)

Bu proje **Vercel** ile entegre çalışmaktadır. GitHub `main` branch'ine yapılan her push işlemi otomatik olarak canlıya (production) aktarılır.
